## 2. 🔍 종합소득세 진단 내부 핵심 기능 (상세 표)

| 구분 | 프리랜서 (3.3% 사업소득) | N잡러 (근로소득 + 사업소득) |
| :--- | :--- | :--- |
| **핵심 목적** | 내년 5월, **돌려받을까 토해낼까?** | 부업 때문에 **세금/건보료 폭탄 맞을까?** |
| **입력/연동 데이터** | 소득 달력에 기록된 '월별 수입 누적액' | 연봉(직장) + 소득 달력의 '부수입 누적액' |
| **계산 엔진 (로직)** | 1. 기납부세액(3.3%) 합산<br>2. 단순경비율 vs 기준경비율 자동 비교 | 1. 종합소득 과세표준 합산 시뮬레이션<br>2. 소득세율 구간 (15% ➔ 24%) 상승 여부 |
| **시각화 (대시보드)** | 계기판(Gauge) 형태의 예상 환급/납부액 | 건보료 폭탄(연 2천만원 초과) 게이지바 |
| **리텐션 넛지 (행동)** | "추가 세금과 보험료를 위해 이번 달 수입의 **15%를 미리 비축**해 두세요!" | "부업 수입이 1800만원을 넘었어요. 지출(경비) 처리에 신경 써야 건보료가 안 올라요!" |
| **특화 공제 시뮬** | 노란우산공제 납입액에 따른 환급액 변화 | 카드 공제는 직장으로, 경비 처리는 부업으로 |
