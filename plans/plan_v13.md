# 홈 화면 6단 레이아웃 전체 청사진

사용자 유형별로 6단 카드 배치를 완성합니다. 각 카드의 구현 상태를 표기하여 향후 작업 순서를 명확히 합니다.

> [!NOTE]
> 구현 상태: ✅ 완성 | 🔨 이번에 작업 | 📋 기획만 (추후 구현)

---

## 직장인

| 단 | 카드명 | 핵심 내용 | 상태 |
|---|---|---|---|
| **최상단** | 배너/알림 카드 | 연말정산 팁, 신혼 공제 안내 등 회전 배너 | ✅ |
| **상단** | 월 급여 입력 카드 | 세전 월급여 입력, 지출 기록 알림 주기 설정 | ✅ |
| **상중단** | 세금 진단 카드 | [연말정산 진단] [기록 모두보기] | 🔨 |
| **중단** | 지출 목표 카드 | 신용/체크 황금비율, 25% 문턱 프로그레스, 넛지 가이드 | ✅ |
| **중하단** | 절세 도구 카드 | [신용카드 관리] [현금영수증 관리] | 📋 |
| **최하단** | FAQ 카드 | 직장인 자주 묻는 질문 3~5개 | 🔨 |

### 직장인 FAQ 후보
1. 중도 퇴사자 연말정산은 어떻게 하나요?
2. 월세 세액공제는 집주인 동의가 꼭 필요한가요?
3. 언제부터 체크카드를 써야 유리한가요?
4. 연말정산을 놓쳤어요. 5월에라도 할 수 있나요?
5. 부양가족 공제, 형제자매도 가능한가요?

## 프리랜서

| 단 | 카드명 | 핵심 내용 | 상태 |
|---|---|---|---|
| **최상단** | 배너/알림 카드 | 5월 종소세 D-day, 노란우산 가입 안내 등 회전 배너 | ✅ |
| **상단** | 수입 입력 카드 | 누적 수입(3.3%), 업종코드, 노란우산공제 | ✅ |
| **상중단** | 세금 진단 카드 | [종합소득세 진단] [기록 모두보기] | 🔨 |
| **중단** | 경비 관리 카드 | 필요경비 증빙 현황, 경비율 구간별 절세 넛지 | 📋 |
| **중하단** | 프리랜서 도구 카드 | [간편장부 작성] [적격증빙 체크리스트] | 📋 |
| **최하단** | FAQ 카드 | 프리랜서 자주 묻는 질문 3~5개 | 🔨 |

### 프리랜서 FAQ 후보
1. 3.3% 떼고 받았는데 5월에 세금을 또 내야 하나요?
2. 단순경비율과 기준경비율 차이가 뭔가요?
3. 식대, 교통비도 경비로 인정받을 수 있나요?
4. 프리랜서도 부가세 신고를 해야 하나요?
5. 종소세 신고를 안 하면 어떻게 되나요?

## N잡러

| 단 | 카드명 | 핵심 내용 | 상태 |
|---|---|---|---|
| **최상단** | 배너/알림 카드 | 투잡 세금 폭탄 경고, 건보료 인상 주의 등 회전 배너 | ✅ |
| **상단** | 소득 합산 입력 카드 | 월급여 입력 + 부업 수입(3.3%) 입력 (두 영역 병렬 표시) | ✅ |
| **상중단** | 세금 진단 카드 | [연말정산 진단] [종합소득세 진단] [기록 모두보기] | 🔨 |
| **중단** | 지출/경비 분리 카드 | 직장 소득공제용 지출(25% 문턱) + 부업 필요경비 분리 모니터링 | 📋 |
| **중하단** | N잡러 도구 카드 | [회사 모르게 종소세 신고법] [부업 장부 작성] | 📋 |
| **최하단** | FAQ 카드 | N잡러 자주 묻는 질문 3~5개 | 🔨 |

### N잡러 FAQ 후보
1. 부업 수입이 생기면 회사에 자동으로 통보되나요?
2. 직장 연말정산 끝냈는데 5월에 종소세도 해야 하나요?
3. 직장인 신용카드 공제 내역을 부업 경비와 중복 처리 가능한가요?
4. 부업 수입이 얼마부터 건보료가 오르나요?
5. 사업자등록 없이 프리랜서 소득 신고가 되나요?

## 이번 작업 범위 (🔨)

위 청사진에서 이번에 구현할 항목만 추출하면 다음과 같습니다:

1. **세금 진단 카드** (상중단) — 기존 메인 메뉴 카드 하단에서 분리하여 독립 카드로 신설
2. **FAQ 카드** (최하단) — 유형별 질문 리스트를 펼침/접힘(Expandable) 형태로 구현
3. **📋 카드들은 "준비 중" 상태로 카드 껍데기만 배치** — 탭 시 "곧 만나요!" 안내

> 📋 상태인 카드들은 레이아웃 자리만 잡아두고, 내부 기능은 추후 별도 작업에서 구현합니다.
> 이렇게 해야 전체 흐름을 미리 확인하면서도 미구현 기능에 시간을 낭비하지 않습니다.
