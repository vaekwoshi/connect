# 종합소득세 진단 기능 구현 계획 (v4 청사진 기준 수정본)

사용자님의 뼈아픈 지적 감사합니다. 제가 이전 버전의 기획(비밀 신고 가이드 등)에 갇혀, 청사진 v4에서 합의했던 내용과 PRD v2의 핵심을 제대로 반영하지 못했습니다. 
'비밀 신고 가이드'를 완전히 배제하고, 순수하게 **PRD v2와 청사진 v4에서 정의한 '종합소득세 진단'의 본질**에 맞춰 기획안을 전면 수정했습니다.

## 1. 종합소득세 진단 - 핵심 기능 (유형별)

### A. 직장인 (근로소득자)
직장인에게는 사실상 '연말정산 진단'이 메인이므로, 종합소득세 진단은 **[N잡러 전환 및 경정청구]**에 초점을 맞춥니다.
- **연말정산 누락분 5월 확정신고(경정청구) 가이드**: 연말정산 때 미처 제출하지 못한 기부금, 안경구입비 등을 5월 종합소득세 신고 기간에 홈택스에서 직접 추가하여 환급받는 시뮬레이션 제공. (비밀 신고 개념 완전 삭제)
- **부수입(N잡) 발생 시뮬레이션**: "만약 올해 부업으로 OOO원을 번다면?" -> 소득 합산 시 세금이 어떻게 변하는지 미리 보여주는 넛지 도구.

### B. 프리랜서 (3.3% 사업소득자) - 가장 핵심 대상
PRD v2에 명시된 **'예상 세액 시뮬레이터 및 세금 비축 넛지'**를 완벽하게 구현합니다.
- **3.3% 환급/납부액 정밀 진단**: 입력된 월별 수입을 누적하여 '추정 소득금액 -> 과세표준 -> 산출세액'을 계산. 이미 떼인 3.3% 원천징수 세액과 비교하여 5월에 **얼마를 돌려받을지(또는 토해낼지)** 진단.
- **세금 및 보험료 비축 넛지 (PRD v2)**: 토해낼 세금, 국민연금(지역가입 9%), 건보료 등을 대비하여 매월 수익의 **최소 15~20%를 따로 저축하도록 안내**하는 넛지 제공.
- **최적 신고유형 및 공제 추천**: 단순경비율 vs 기준경비율 유리함 비교, 노란우산공제 납입 시 환급액 상승폭 시뮬레이션.

### C. N잡러 (근로소득 + 사업소득)
- **합산 과세표준 및 세율 점프 진단**: 직장 연봉과 부수입을 합쳤을 때 소득세율 구간이 상승하는지 진단하여 절세 타이밍 안내.
- **건강보험료 폭탄 방어 (PRD v2)**: 부업 소득이 연 2,000만 원을 초과하여 '소득월액 보험료'가 추가 부과되기까지 얼마 남았는지 게이지바로 시각화.

## 2. 세무 도구 카드 UI 변경 계획 (홈 화면 연동)
현재 `home_screen.dart`의 도구 카드에서 잘못 남아있는 서브타이틀과 연결 로직을 수정합니다.

- **직장인 모드**: '놓친 공제·민감 항목을 따로 환급받아요' -> **'연말정산 누락분, 5월에 추가로 돌려받기'**로 텍스트 수정.
- **프리랜서/N잡러 모드**: 진단 탭 터치 시, 복잡한 입력 없이 '나의 세금 기록부(월별 수입/지출)'에 입력된 데이터를 즉시 끌어와서 환급/납부액 계기판(Dashboard) 뷰를 띄워줌.
