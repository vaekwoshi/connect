# 홈 화면 5단 레이아웃 전체 청사진 v4.2

> [!NOTE]
> 사용자 지시 반영:
> 1. '진단' 명칭 삭제 및 '나의 세금 기록부' 통합 폐지 ➔ **[연말정산]**, **[종합소득세]** 2개 탭으로 깔끔하게 통폐합.
> 2. 유형/절세 기준 판별 문진(질문)은 **최상단 카드**로 이동하여, 응답 시 앱 모드가 자동 전환되도록 로직 변경.

---

## 직장인

| 단 | 카드명 | 핵심 내용 | 상태 |
|---|---|---|---|
| **최상단** | 배너/알림 카드 | **[유형 판별 질문]** 타 소득(부업, 금융 등) 유무 질문 ➔ 응답 시 N잡러 모드 자동 전환 | 📋 |
| **상단** | 월 급여 카드 | 세전 월급여 입력 / 하위: **[소득 기록 모두보기 (달력 뷰)]** | ✅ |
| **중단** | 지출 목표 카드 | 신용/체크 황금비율, 25% 문턱 프로그레스, 넛지 가이드 | ✅ |
| **중하단** | 도구 카드 | **[연말정산]**, **[종합소득세]** | 📋 |
| **최하단** | FAQ 카드 | 직장인 자주 묻는 질문 (아코디언 드롭다운 적용) | ✅ |

**직장인 도구 카드 상세**:
| 도구 | 설명 | 상태 |
|---|---|---|
| 연말정산 | 당해 연도 연말정산 시뮬레이션 + 과거 연말정산 기록부 모아보기 | 📋 |
| 종합소득세 | 당해 연도 종소세 시뮬레이션 + 과거 종소세 기록부 모아보기 | 📋 |

## 변경된 작업 범위 (📋)

1. **최상단 카드 로직 개편**: 타 소득/부업 유무를 묻는 팝업을 띄우고, 결과에 따라 사용자 앱 모드(Provider/State) 자동 전환 로직 구현
2. **도구 카드 통합 UI 개발**: 기존 진단/기록부 UI를 갈아엎고 `연말정산` 및 `종합소득세` 2개의 깔끔한 버튼으로 통합 (클릭 시 하위에서 시뮬레이터/기록부 조회)
