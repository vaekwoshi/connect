# 홈 화면 5단 레이아웃 전체 청사진 v4

> [!NOTE]
> 구현 상태: ✅ 완성 | 🔨 이번에 작업 | 📋 기획만 (추후 구현)

> [!IMPORTANT]
> v3 대비 변경점:
> - **세금 진단 카드(상중단) 제거** — 자체 데이터가 없는 단순 메뉴 껍데기였음
> - **연말정산·종소세 진단 → 도구 카드로 이동** — 시뮬레이션은 도구의 일종
> - **기록 모두보기 → 소득 카드 하위 메뉴로 이동** — 누적 소득·과거 기록과 맥락 일치
> - 6단 → **5단 구조**로 축소하여 스크롤 최적화

## 직장인

| 단 | 카드명 | 핵심 내용 | 상태 |
|---|---|---|---|
| **최상단** | 배너/알림 카드 | 연말정산 팁, 신혼 공제 안내 등 회전 배너 | ✅ |
| **상단** | 월 급여 카드 | 세전 월급여 입력 / 하위: **[모두보기]** (누적 급여 이력, 과거 시뮬레이션 기록) | ✅ → 🔨 |
| **중단** | 지출 목표 카드 | 신용/체크 황금비율, 25% 문턱 프로그레스, 넛지 가이드 | ✅ |
| **중하단** | 도구 카드 | [연말정산 진단] [종합소득세 진단] [비밀 신고 가이드] | 🔨 |
| **최하단** | FAQ 카드 | 직장인 자주 묻는 질문 | 🔨 |

**직장인 도구 카드 상세**:

| 종합소득세 진단 | 민감 공제 비밀 신고, 경정청구, 추가소득 합산 | 🔨 |
