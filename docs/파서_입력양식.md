## E. 확인된 실제 구조 (2025 귀속 샘플 분석 결과)

받은 직장인 샘플 2개를 분석한 결과 — **둘 다 PDF 텍스트가 깔끔히 추출됨**(이미지 아님). 온디바이스 텍스트 파싱 확정 가능. 한국 정부 PDF지만 글꼴에 유니코드 매핑이 있어 추출 OK.

### E-1. 연말정산 간소화 자료 (PDF) — "공제 가능액"
- **구조:** 카테고리 1개 = 페이지 1개(이상). 각 페이지 머리글: `2025년 귀속 소득·세액공제증명서류 ... [카테고리]`
- **파싱법:** `[카테고리]` 키워드로 섹션 분할 → 각 섹션의 **합계 라인** 추출.

| 카테고리 머리글 | 합계 라인 라벨 | → 앱 필드 |
|---|---|---|
| `[건강보험료]` | `총합계` | 4대보험(`insuranceDeduction`) |
| `[고용보험료]` | `합계` | 〃 |
| `[국민연금보험료]` | `총합계` | 〃 |
| `[보장성 보험…]` | `인별합계금액` (한도 100만) | `insurancePremium` |
| `[의료비]` | `인별합계금액` | 의료비(− 실손) |
| `[실손의료보험금]` | `인별합계금액` | 의료비에서 차감 |
| `[신용카드]` | 집계행 `일반 전통시장 대중교통 문화체육 합계금액` | `creditCard`(합계금액) |
| `[직불카드 등]` | 〃 | `debitCard` 일부 |
| `[현금영수증]` | 〃 | `debitCard` 일부 |
| `[교육비]`·`[기부금]`·`[연금저축]`·`[주택자금]`·`[월세액]` | (해당 지출 있을 때만 페이지 생성) | 각 공제 필드 |

> 주의: 지출이 없는 카테고리는 **페이지 자체가 없음** → 파서는 "없으면 0"으로 처리. (샘플엔 교육비·기부금·연금저축·월세 페이지 없음)
> 카드 raw 사용액(예: 신용 3,712,061)을 그대로 넣으면 됨 — 25% 문턱·공제율은 앱이 계산.

### E-3. ★ 핵심 발견 — "빠진 공제"는 두 파일의 차이
- **간소화 = 공제 가능액**, **원천징수영수증 = 실제 신고액(62 의료비, 63 교육비, 64 기부금, 70 월세 …)**.
- **빠진 공제 = 간소화(가능) − 원천징수영수증(신고됨)** → 그 차액이 경정청구/5월 환급 대상.
- 즉 ②진단을 처음부터 재계산할 필요 없이, **두 PDF를 대조**해 "안 넣은 공제"를 바로 찾을 수 있음. (샘플 인물은 의료비 148,400이 총급여 3% 문턱 미달이라 실제 누락은 없었음 — 로직 검증됨)

### E-4. HWP 빈 양식 분석 결과 (별지23(3)·별지24)
받은 **HWP 별지 서식**(빈 템플릿)도 olefile+zlib로 추출됨. 단 **값이 없는 구조 템플릿**이고, 실사용자는 HWP가 아니라 PDF/XLSX를 내려받으므로 **앱 파서 대상은 아님** — 다만 필드 레이아웃 확인엔 유용했음.

**[별지23(3)] 사업소득 원천징수영수증·지급명세서(연말정산용)** 핵심 필드(프리랜서 파서용):
- **지급액(수입금액)** → `freelancerIncome`
- 보험모집/방문판매/음료배달 수입금액 계, **합계(124)**
- 소득금액, **소득세(원천징수세액)** → 기납부, 농어촌특별세
- 결정세액, 차감 납부할 세액, 인적·자녀·연금·기부금 공제
> ※ 이 서식은 **연말정산 대상 사업소득**(보험모집인·방문판매·음료배달)용. 일반 인적용역 프리랜서(3.3%)는 **거주자 사업소득 원천징수영수증/지급명세서** 또는 **소득금액증명**이 더 적합.

### E-5. 결론 — 더 받을 것 / 바로 될 것
- ✅ **직장인 파서: 지금 받은 PDF 2개로 바로 구현 가능** (간소화 + 원천징수영수증, 둘 다 실값·파싱 검증됨).
- ⛔ **추가 HWP 빈 템플릿은 불필요** — 의료비·교육비·기부금·연금저축·신용카드 개별 증명서는 모두 **간소화 자료 안에 번들**되어 들어옴(이미 1부 확보).
- 🔶 **프리랜서까지 하려면 "값이 채워진" PDF 1부 필요**: 사업소득 원천징수영수증/지급명세서 **또는** 소득금액증명 (더미값 OK). HWP 템플릿 말고 실제 내려받은 PDF로.
- 🔶 (선택) 교육비·기부금·연금저축·월세가 **실제로 있는** 간소화 자료 1부 → 그 카테고리 합계 라벨 최종 확정.
