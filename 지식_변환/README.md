# 🧠 세끌 프리미엄 세무 지식 통합 보관소 (Sekkeul Tax Knowledge Base)

이 폴더는 세무사 검증을 거친 새로운 진짜 세법 지식 자료들을 종류별로 안전하고 정갈하게 보관하는 세끌 프로젝트의 공식 통합 브레인 저장소입니다.

---

### 📂 파일 종류별 정밀 분류 구조

---

여기에 지식 자료가 추가되면, 젬(Gemini)이 즉시 정밀 분석하여 `js/app.js` 세무 엔진에 100% 무오류로 자동 이식해 드립니다!

---

### 🛠️ 젬 전용 `/지식` 명령어 규칙 (Knowledge Isolation Protocol)

젬(Gemini)은 사용자가 `@젬 /지식 <질문>` 형태로 명령어를 던지면 즉시 **"지식 격리 프로토콜"**을 의무 가동합니다.

1. **인터넷 검색 금지**: 외부 웹 검색(`search_web` 등)을 일절 수행하지 않습니다.
2. **외부 일반 지식 배제**: 모델 내부의 범용 세법 지식을 섞지 않고 오직 `지식/` 폴더 내 폴더(`지식_변환/`)에 적힌 것을 유일한 진실(Ground Truth)로 삼습니다.
3. **엄격한 거부 규칙**: 질문에 대한 근거가 지식 폴더 파일 내에 존재하지 않거나 모호할 경우, 절대 임의로 상상(Hallucination)하여 답하지 않고 **"해당 내용은 세끌 지식 폴더 내에 기재되어 있지 않은 정보입니다."** 라고 안전하게 답변을 제한합니다.
