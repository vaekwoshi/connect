# 2025년 귀속 연말정산 정답지 + 개정사항 + 엔진 검증

> 출처: 국세청 「2025 원천징수의무자를 위한 연말정산 신고안내」 (요약 p4~8, 개정 p19~25, 정오표).
> 수치 단일기준(SSOT)은 `JSON/2025_연말정산_공제율_정답지.json`. 이 MD는 사람이 읽는 요약 + 앱 엔진 검증.

## 계산 흐름 (전체 골격)
```
연간 근로소득 - 비과세 = 총급여액
총급여액 - 근로소득공제 = 근로소득금액
근로소득금액 - 인적공제 - 연금보험료공제 - 특별소득공제 - 그밖의소득공제 (+종합한도초과) = 과세표준
과세표준 × 기본세율 = 산출세액
산출세액 - 세액감면 - 세액공제 = 결정세액
결정세액 - 기납부세액 = 환급 / 추가납부
```

## 2025년 주요 개정 (꼭 반영)
1. **자녀세액공제 확대**: 첫째 15→**25만**, 둘째 20→**30만**, 셋째이후 30→**40만**/인. (8세이상 자녀·손자녀)
2. **고향사랑기부금 한도 500만→2,000만** (정오표 정정).
3. 종업원 할인금액 근로소득 비과세 신설: Max(시가 20%, 240만).
4. 연금계좌 추가납입 대상 확대(1주택 고령가구 다운사이징 차액 등, 생애 1억 한도). 납입한도 1,800만 동일.
5. 장애인 범위 명확화 + 발달재활서비스 이용증명서 증빙 인정.
6. 퇴직자 노조회비 세액공제 요건 완화.

## 핵심 한도 빠른참조
- **근로소득공제 한도**: 2,000만원
- **인적공제 기본**: 1인 150만 / 경로 100 / 장애 200 / 부녀 50 / 한부모 100
- **신용카드 한도**: 7천이하 300만 / 7천초과 250만 / 1.2억초과 200만
- **연금계좌**: 연금저축 600만, IRP합산 900만 / 12%(5500만↓ 15%)
- **의료비**: 총급여 3% 초과분, 일반 700만, 난임30%·미숙아20%·그외15%, 산후조리원 200만
- **교육비**: 취학전·초중고 300만, 대학 900만, 본인·장애인 전액 (모두 15%)
- **월세**: 750만→**1000만 한도**, 17%(5500만↓)/15%
- **기부금**: 일반·특례 15%(1천만초과 30%), 정치자금 10만까지 100/110, 고향사랑 2000만 한도
- **소득공제 종합한도**: 2,500만 초과분 과표 합산
