## 1. 계산엔진이 "안 되는" 항목 (우선순위순)

### 🔴 최우선 — 정확도에 큰 영향, 거의 모든 사용자 해당
1. **연금보험료공제(국민연금) + 특별소득공제 보험료(건강·고용)**
   - 현재: 4대보험 계산(`calculateMonthlyInsurance`)은 있으나 **연말정산 과세표준 차감에 미연동**.
   - 영향: 과세표준이 실제보다 높게 잡혀 결정세액 과대·환급 과소.
   - 해결: 기존 `InsuranceBreakdown` 값을 연 환산해 과표에서 빼기만 하면 됨. **신규 엔진 불필요**.

2. **자녀세액공제** — 엔진에 아예 없음
   - 자녀 있는 사용자 전원 해당. 현재 0원 처리 → 환급액 과소.
   - 2025 귀속 기준: 8세이상 1명 15만 / 2명 35만 / 3명이상 1명당 +30만, 출산·입양 첫째30·둘째50·셋째이상70만.

3. **연금계좌 세액공제(연금저축 + IRP)** — 홈 세무도구에 "준비중" 표기된 항목
   - 직장인 절세 **1순위** 항목. 전용 계산기 필요.
   - 2025 귀속: 연금저축 연600만 / IRP 합산 연900만 한도, 12%(총급여 5500만 이하 15%).

### 🟡 중간 — 해당자에게 큰 금액
4. **보장성보험료 세액공제** (보장성 100만 12% / 장애인전용 100만 15%) — 없음
5. **교육비 확장** — 본인 교육비(전액·무제한), 취학전아동(300만), 장애인 특수교육비(전액) 누락. 현재 초중고/대학만.
6. **기부금 유형 분리** — 정치자금(10만까지 100/110), **고향사랑기부금(2025 신설)**, 우리사주, 이월분 미지원. 현재 단일 15%/30%만.
7. **주택자금** — 주택임차차입금 원리금(40%) 없음, 장기주택저당 이자는 단순 2000만 한도만(차입연도/상환조건 분기 없음).

### ⚪ 후순위 — 사용 빈도 낮음
8. 노란우산(소기업소상공인공제부금, 프로필 필드만 존재), 주택마련저축, 투자조합출자, 우리사주출연금, 장기집합투자증권저축, 외국납부세액, 표준세액공제(13만).

## 2. 현재 계산과 "기준이 다른" 항목

| 항목 | 앱 엔진 | 2021 양식 | 판정 |
|---|---|---|---|
| 월세 세액공제 | 17%/15%, **1000만 한도** | 12%/10%, 750만 한도 | **앱이 최신**(2024 귀속 개정 반영) ✅ |
| 난임시술비 | **30%** | 20% | **앱이 최신** ✅ |
| 의료비 미숙아·선천성이상아 | 분류 없음(15% 처리) | — | 20% 별도 카테고리 누락 ⚠️ |
| 기부금 | 15% / 30% | 20% / 35%(양식 표기) | **최신 법령 재확인 필요** ❓ |
| 고향사랑기부금 | 없음 | 2025 서식 신설 | 누락 ⚠️ |

> 결론: 월세·난임은 앱이 맞고 2021 양식이 구버전. 기부금 요율과 고향사랑기부금만 최신 법령 교차확인이 필요합니다. **인터넷 수치 금지 원칙**에 따라, 이 두 가지는 사용자가 보유한 최신 국세청 자료(아래 3번)로 확정해야 합니다.

## 3. 더 필요한 자료 (사용자에게 요청)

현재 4종으로는 **요율·한도 수치 확정**이 부족합니다. 다음이 있으면 엔진을 정확히 맞출 수 있습니다:

1. **「연말정산 신고안내」 책자 본문** (이미 바탕화면에 `2025년 원천징수의무자를 위한 연말정산 신고안내` 폴더 보유 → 그 안의 본문 HWP/PDF). ← **가장 중요**. 항목별 최신 공제율·한도 표가 여기 있음.
2. **자녀세액공제 / 연금계좌 세액공제 상세 규정** (위 책자 내 해당 챕터).
3. **기부금 세액공제율 최신표** (특례/일반/고향사랑/정치자금 구분, 2025 귀속).
4. (선택) **종합소득세 신고서식 본체**(별지40호 종합소득세 과세표준확정신고서) — 5월 신고서 양식 출력 정확도용.

> 위 자료는 모두 **내부 지식 자료**로 보유 중이거나 국세청 공식 책자라, 외부 인터넷 수치 없이 엔진 보강이 가능합니다.

## 4. UI 카드 배치 권고 — "사용자는 적게 입력, 앱이 자동 계산"

설계 원칙: **자동 계산 가능한 건 묻지 않는다.** 사용자 입력은 증빙이 필요한 실지출 항목만.

### A. 프로필 / 자동 (입력 0~최소)
- 4대보험·연금보험료공제 → 총급여로 자동 계산(엔진 재활용)
- 인적공제 기본/추가, **자녀세액공제** → 프로필의 부양가족·자녀 수·연령에서 자동
- 근로소득공제·근로소득세액공제·표준세액공제 → 전부 자동

### B. 홈 "지출 카드"(이미 있음)
- 신용카드등 소득공제 → 지출 달력과 이미 연동. 유지.

### C. 연말정산 wizard(이미 구현) — 확장
- 의료비/교육비/기부금/월세 = **실지출 증빙 항목** → 현재 4단계 wizard 유지.
- 확장: 교육비에 본인·취학전·장애인 추가, 기부금에 유형 분리, 보장성보험료 단계 추가.

### D. 신규 전용 카드 — "연금저축·IRP 절세 계산기"
- 절세 1순위 항목이라 독립 카드로. "연 OOO만원 넣으면 OO만원 환급" 식 즉시 시뮬레이션.
- 입력 1개(연 납입액) + 나이(50세 기준 한도 분기) → 전용 엔진.

### 간단 이용 흐름(권고)
```
프로필(1회 입력: 부양가족·자녀·월세 여부)
   → 홈에서 소득·지출 자동 누적
   → 연말정산 진단(자동 계산 + wizard로 실지출만 추가)
   → "종합소득세 신고서 준비"(5월 경정청구)
```
사용자가 직접 숫자를 넣는 곳은 **wizard의 실지출 4~6개 항목**과 **연금저축 납입액 1개**뿐이 되도록.
