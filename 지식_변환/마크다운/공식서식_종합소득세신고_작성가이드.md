# 종합소득세 신고 공식서식 완벽 작성 가이드

> 출처: 국세청 2025년 개정 공식서식 (별지 24호, 37호 등)  
> 대상: 세끌 `tax_annual_report_screen` 사용자 및 직장인 + N잡러  
> 목적: HomeTax 신고 전 필수 체크리스트 및 단계별 입력 안내

---

## 📋 핵심: 회사에 제출할 서류 vs. HomeTax에 입력할 서류

| 서류명 | 대상 | 제출처 | 형식 | 기한 |
|--------|------|--------|------|------|
| **별지37: 소득·세액 공제신고서** | 근로자 | 회사(원천징수의무자) | 종이/전자 | 연말정산 마감 전 |
| **별지24: 원천징수영수증** | 근로자 | 근로자 보관 | 발급자(회사) 발행 | 연말정산 후 |
| **별지45: 기부금명세서** | 필요시 | 회사 | 종이 | 별지37과 동시 |
| **별지43: 의료비명세서** | 필요시 | 회사 | 종이 | 별지37과 동시 |
| **기타 증명서류** | 필요시 | 회사 | 원본/사본 | 별지37과 동시 |

## 1️⃣ 별지37 소득·세액 공제신고서 작성 단계

### 1단계: 기본 정보 입력
```
- 소득자 성명: 이름 정확히 기입 (외국인: 로마자 표기)
- 주민등록번호: XXXXXX-XXXXXXX 형식
- 근무처 명칭: 회사 정식명칭 (약칭 불가)
- 사업자등록번호: XXX-XX-XXXXXX 형식
- 세대주 여부: 세대주/세대원/세대주의 배우자 중 선택
- 거주구분: 거주자(일반) / 비거주자(외국인 단일세율)
```

### 2단계: 인적공제 입력 (별지37 1쪽)

**기본공제 대상자 작성**
1. 본인 (관계코드 0): 필수
2. 배우자 (관계코드 3): 소득기준 확인
   - 소득금액 100만원 이하 (근로소득만 있으면 총급여 500만원 이하)
3. 부양가족 (관계코드 1~8): 각 가족마다 줄 추가
   - 나이 요건 확인 (60세이상 직계존속, 20세이하 직계비속 등)
   - 소득기준 확인

**추가공제 항목 (있는 경우만)**
- 경로우대: 70세 이상 부양가족 ✓ → 100만원 추가
- 장애인: 장애인증명서 보유 ✓ → 200만원 추가
- 부녀자: 여성+부양가족 있음 ✓ → 50만원 추가
- 한부모: 배우자 없음 + 부양가족 있음 ✓ → 100만원 추가

### 3단계: 소득·세액 공제 항목 입력

#### A. 보험료 공제 (소득공제)
**국민연금, 건강보험, 고용보험** (간소화 자동 제공)
- 종(전)근무지 / 주(현)근무지 분리 입력
- 전액 공제 (한도 없음)

#### B. 특별소득공제 (소득공제)
**주택임차차입금** (직접 입력)
- 필수서류: 임대차계약서, 계좌이체 확인서 12개월치
- 공제율: 원리금상환액의 40%
- 한도: 연 400만원

#### C. 세액공제 항목

**혼인세액공제** (있는 경우만)
- 2024~2026년 혼인신고자
- 금액: 50만원 (생애 1회)
- 증명서: 혼인관계증명서

**자녀세액공제** (기본공제 대상 자녀)
- 8세 이상만 (해당과세기간 중 8세 도달 포함)
- 1명: 25만 / 2명: 55만 / 3명+: 55만+2명초과×40만
- 별도 서류 불필요 (기본공제 대상자면 적용)

**연금계좌 세액공제** (필수 서류: 납입확인서)
| 총급여 | 세액공제 대상 | 공제율 |
|--------|-------------|--------|
| 5,500만원 이하 | 연금저축 600만 + IRP 900만(합산) | 15% |
| 5,500만원 초과 | 연금저축 600만 + IRP 900만(합산) | 12% |

**보장성보험료 세액공제** (필수 서류: 보험료납입증명서)
- 일반 보장성: 납입액 × 12%, 한도 100만원
- 장애인 전용: 납입액 × 15%, 한도 100만원
- 주의: 피보험자가 기본공제 대상자여야 함

**의료비 세액공제** (필수 서류: 의료비영수증)
- 자동 제공 + 직접 입력 (간소화 미조회분)
- 공제 공식: (의료비 총액 - 총급여×3%) × 공제율
- 유형별 공제율:
  - 일반의료비: 15% (한도 700만)
  - 본인·65세이상·6세이하·장애인: 15% (무제한)
  - 미숙아·선천성이상아: 20% (무제한)
  - 난임시술비: 30% (무제한)
  - 산후조리원: 15% (한도 200만원/출산 1회)

**교육비 세액공제** (필수 서류: 교육비납입증명서)
- 공제율: 15% (무제한)
- 대상별 한도:
  - 취학전 아동, 초·중·고등학생: 1명당 300만원
  - 대학생: 1명당 900만원
  - 대학원: 불가
  - 본인, 장애인 특수교육비: 무제한
- 주의: 취학전 학원비만 O, 초등학생 학원비는 X

**신용카드 등 소득공제** (자동 제공)
- 계산 공식: (신용카드 사용액 - 총급여×25%) × 공제율
- 공제율: 신용카드 15%, 체크·현금 30%, 문화체육 30%, 전통시장 40%, 대중교통 40%
- 한도: 7,000만원 이하 기본 300만+추가 300만, 7,000만원 초과 기본 250만+추가 200만

**월세액 세액공제** (필수 서류: 임대차계약서, 계좌이체 확인서)
- 조건: 무주택/1주택 세대주, 총급여 8,000만원 이하
- 공제율: 총급여 5,500만원 이하 17%, 5,500~8,000만원 15%
- 한도: 연 1,000만원
- 대상 주택: 아파트, 오피스텔, 고시원, 기준시가 4억원 이하

**기부금 세액공제** (필수 서류: 기부금영수증)
- 정치자금: 10만원 이하 100/110, 초과 15%, 3,000만원 초과 25%
- 고향사랑기부금: 10만원 이하 100/110, 초과 15% (한도 2,000만원)
- 특례기부금(국방헌금 등): 1,000만원 이하 15%, 초과 30%
- 일반기부금(종교·문화 등): 1,000만원 이하 15%, 초과 30% (한도: 근로소득금액×30%)
- **이월공제**: 특례·일반기부금은 10년 이월 가능, 정치자금·고향사랑은 불가

## 2️⃣ 필요한 증명 서류 체크리스트

| 항목 | 서류명 | 조회처 | 미조회시 대응 |
|------|--------|--------|-------------|
| 4대보험 | 간소화 자료 | HomeTax | 회사 문의 |

## 3️⃣ 별지24 원천징수영수증 확인 항목

회사에서 발급하는 이 서류가 정확한지 꼭 확인하세요!

**확인 항목**
1. 총급여액 = 급여 + 상여 + 기타 수입
2. 결정세액 = 올바른 공제 후 계산된 세액
3. 기납부세액 = 월별로 회사에서 원천징수한 누적 세액
4. 환급액 또는 추가납부액 = 결정세액 - 기납부세액

**오류 발견 시**
- 회사(원천징수의무자)에 즉시 신고
- 정산 후 정정 지급명세서 발급 요청
- 5년 내 경정청구 가능

## 4️⃣ HomeTax 신고 전 최종 체크

### 신고 전 준비물
- [ ] 별지37 소득·세액 공제신고서 (회사 제출용)
- [ ] 간소화 자료 (신용카드, 보험료, 의료비, 교육비 등)
- [ ] 부양가족 자료제공 동의 완료 (부양가족 공제받을 경우)
- [ ] 의료비/교육비/기부금 영수증 (간소화 미조회분)
- [ ] 연금계좌 납입확인서
- [ ] 보장성보험료 납입증명서
- [ ] 월세 관련 서류 (임대차계약서, 계좌이체 확인서)
- [ ] 기부금 영수증

### 신고 순서
1. HomeTax에 공제신고서 작성 시작
2. 간소화 자료 자동 반영
3. 기타 공제 항목 수동 입력
4. 세액 자동 계산 확인
5. 최종 검토 후 제출
6. 환급/추가납부 확인

### 오류 시 대응
- **연말정산 후 오류 발견**: 경정청구 (5년 이내, 홈택스)
- **세액공제 누락**: 5월 종합소득세 신고에서 추가 공제 가능

## 5️⃣ 특수 상황별 작성 팁

### 👔 직장인 + 부업(사업소득)
- N잡러로 분류됨 → 합산소득세 적용
- 5월 종합소득세 신고 의무
- 사업소득 추가 공제: 단순경비율 또는 기장 기준

### 🏠 무주택 세대주 (월세)
- 월세액 세액공제 17% (5,500만원 이하) 또는 15%
- 신용카드 소득공제와 중복 불가 → 더 유리한 것 선택
- 필수: 임대차계약서 + 12개월 계좌이체 증명

### 👨‍👩‍👧 부양가족 많은 경우
- 인적공제: 1인 150만원 (기본) + 100~200만원 (추가)
- 자녀세액공제: 8세이상만 (25~40만원/인)
- 부양가족이 다른 세대주(배우자 등)에게 공제받으면 중복 불가

### 🎓 대학생 자녀
- 교육비 한도: 1명당 연 900만원
- 학자금대출: 교육비 납입 연도가 아닌 **상환 연도에만 공제** (논문 이상)
- 대학원: 본인만 전액 공제 가능

## ⚠️ 자주 하는 실수

| 실수 | 원인 | 해결책 |
|------|------|--------|
| 배우자 중복 공제 | 부부가 각각 공제신고 | 한쪽만 기본공제 신청 |
| 월세 + 신용카드 중복 | 월세도 신용카드로 결제 | 월세액 공제 선택 (유리한 것) |
| 부양가족 소득기준 초과 | 소득 100만원 초과 체크 안함 | 부양가족 소득 사전 확인 |
| 의료비 3% 문턱 간과 | 3% 미만은 공제 안됨 | 총급여 × 3% 계산 후 입력 |
| 교육비 대학원 포함 | 대학원도 공제 가능하다고 착각 | 대학원은 본인만, 자녀 불가 |
| 미제출 의료비 | 간소화에 없는 항목 누락 | 의료기관 직접 수집 |
