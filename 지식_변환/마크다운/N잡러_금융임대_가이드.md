# N잡러 (투잡 직장인, 금융소득, 다주택자) 가이드
## 1. 직장가입자 소득월액 보험료 추가 부과
직장을 다니고 있더라도 월급을 제외한 보수외 소득(부업 사업소득 + 금융소득 + 주택임대소득 등)의 합계가 **연 2,000만 원을 초과**하면 월급 건보료 외에 추가 건보료가 청구됩니다. (초과분에 대해 7.19% 부과. 본인 100% 부담)

## 2. 소득금액 환산 가이드 (2026 최신)
- **금융소득(이자/배당):** 연 1,000만 원 초과 시 전액 100% 소득으로 합산됩니다.
- **주택임대소득:** 
  - 등록 임대사업자: (임대수입 × 40%) - 400만 원
  - 미등록 임대사업자: (임대수입 × 50%) - 200만 원
  *(단, 임대소득 외 소득이 2천만 원을 넘으면 기본공제 400/200만 원은 적용되지 않습니다)*

## 3. 투잡 직장인 (이중 근로소득자)
낮엔 A회사, 밤엔 B회사에서 모두 근로소득을 받는 경우, 합산 신고를 통해 누진세율을 다시 적용받아야 합니다.

## 4. 간주임대료 (2025 개정)
임대보증금에 대한 간주임대료 계산 기준이자율: **연 3.5% → 3.1%** (2025.3.21 이후 차입분부터)
- 간주임대료 = (임대보증금 - 임대주택 취득에 소요된 차입금) × 3.1% × 임대일수/365
